PROJECTS and SEMINAR

  • PIMA Indian Diabetes Classification:

    • It aims in determining if an individual is diagnosed with Diabetes

    • Eight attributes were used to build the model. Validation accuracy achieved is 75.6%

    • Currently working on improving the accuracy by advanced optimization techniques

    • Technology stack: MATLAB, Logistic Regression, and Stochastic Gradient Descent

  • Breast cancer detection:

    • A preliminary test to check if a person is diagnosed with breast cancer

    • The dataset used has 9 attributes. 60% of the data was used on training and 40% on testing

    • To overcome zero frequency problem, Laplace Smoothing technique was implemented

    • Testing accuracy was 84.5%

    • Technology stack: MATLAB, Naïve Bayes, and Laplace Smoothing

  • Crop yield prediction using machine learning:

    • Final year project - This application predicts yield of the crop based on eight climatic and four soil parameters

    • Implemented polynomial regression with ordinary least squares

    • It has an overall accuracy of 87%, 4.04% more than the existing system

    • Technology stack: MATLAB, Polynomial Regression, and Ordinary Least Squares

  • Seminar: Benchmarking of supervised learning algorithms in IP traffic classification:

    • Demonstrated the classification of IP traffic generated in NFV

    • Presented an analysis of the efficiency of 3 network traffic classifying algorithms based on their response times

    • Technology stack: Network Function Virtualization, Weka, J48, Naïve Bayes, and BayesNet

  • Android Applications

    • Vision – An application used to mimic the functions of the machine learning application developed to predict the crop yield

    • Jazzy – Results in the corresponding color of the given RGB value based on input in an android phone

    • Technology stack: Windows, Linux, Android, XML, and Java

  • Python Mini Projects

    • Tic - Tac - Toe – A console based game which allows 2 users to play the game. It notifies if a player won or if a deadlock has occurred

    • BlackJack – A console based card game that allows 1 player to play against the house (computer) It allows the user to place a bet and accordingly the user may win or lose the bet

    • Word Scrambler – It is a program that reads a file and scrambles the text in the file based on 5 rules and writes the scrambled content into an output file. Provides a feature of encryption

    • Technology stack: Jupyter Lab with Anaconda Navigator and Python
